# Tuesday Daily Assignment - Week 8

## Usage Instructions
*   **Python Target:** Ensure environment covers standard packages (`pandas`, `numpy`, `scikit-learn`, `matplotlib`). Note that PyTorch/Tensorflow is explicitly not required for this assignment constraint.
*   **Data Structure:** All `.csv` datasets (like `train_data.csv`) must be mapped strictly inside an inner **`data/`** directory. Execution will inherently fail if the raw datasets execute alongside the `analysis.ipynb` layout since `load_all_csvs()` searches purely for the localized `data` folder.
*   **Execution:** Launch `analysis.ipynb` and trigger a Kernel `Restart & Run All`. All Data Quality audits, Neural Network epoch convergence tracking, and PCA Embeddings comparisons will inherently compile automatically from top to bottom. 

## Included Files
- `analysis.ipynb`: Comprises the extensive dataframe audit configurations explicitly handling categorical features, alongside a Custom NumPy Matrix back-propagation classification MLP network.
- `observations.md`: Fulfills the analytical documentation task assignments describing the performance of Intermediate Latent Representation logic dynamically extracted from Neural Architectures.
